## Approaches

### ЁЯФ╣ 1. Using a `for` loop (classic way)

```javascript
var filter = function (arr, fn) {
  let res = [];
  for (let i = 0; i < arr.length; i++) {
    if (fn(arr[i], i)) {
      res.push(arr[i]);
    }
  }
  return res;
};
```

---

### ЁЯФ╣ 2. Using `for...of` with index

```javascript
var filter = function (arr, fn) {
  let res = [];
  for (let [i, val] of arr.entries()) {
    if (fn(val, i)) res.push(val);
  }
  return res;
};
```

---

### ЁЯФ╣ 3. Using `forEach`

```javascript
var filter = function (arr, fn) {
  let res = [];
  arr.forEach((val, i) => {
    if (fn(val, i)) res.push(val);
  });
  return res;
};
```

---

### ЁЯФ╣ 4. Using `reduce` (functional style)

```javascript
var filter = function (arr, fn) {
  return arr.reduce((res, val, i) => {
    if (fn(val, i)) res.push(val);
    return res;
  }, []);
};
```

---

### ЁЯФ╣ 5. Using recursion

```javascript
var filter = function (arr, fn, i = 0) {
  if (i >= arr.length) return [];
  const rest = filter(arr, fn, i + 1);
  return fn(arr[i], i) ? [arr[i], ...rest] : rest;
};
```

## Example Usage

## Complexity Analysis

* **Time complexity:** O(n) тАУ each element is visited once.
* **Space complexity:** O(n) тАУ result array stores up to `n` elements.
